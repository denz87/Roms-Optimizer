📖 개요
ROM Cleaner는 ROM 컬렉션을 관리하고 정리하는 고급 도구입니다.
프로그램은 자동으로 중복을 식별하고 지능적인 품질 기준과
지리적 우선순위에 기반하여 최고의 버전만을 유지합니다.

═══════════════════════════════════════════════════════════════════

🔧 주요 기능

📂 폴더 선택
   • ROM이 포함된 메인 폴더 선택
   • 프로그램이 모든 하위 폴더를 재귀적으로 스캔
   • 모든 일반적인 ROM 형식 지원

⚙️ 정리 옵션
   • 테스트 모드: 파일을 수정하지 않고 정리를 시뮬레이션
   • 보관 파일 표시: 보관될 파일도 표시
   • 이동 대신 복사: 원본 파일을 유지하고 복사본 생성

🌍 지리적 우선순위
   • 유럽: 이탈리아, 유럽 및 PAL 버전 우선
   • 미국: 미국 및 NTSC 버전 우선
   • 일본: 일본 및 아시아 버전 우선

═══════════════════════════════════════════════════════════════════

🏆 우선순위 시스템

프로그램은 각 ROM의 최고 버전을 결정하기 위해
고급 점수 시스템을 사용합니다:

📍 지리적 우선순위 (기본):
   • 지리적 태그가 없는 파일: 최대 우선순위
   • 이탈리아/이탈리아어: 1000점
   • 유럽/PAL: 900점
   • 미국/NTSC: 750점
   • 영국/영어: 700점
   • 일본: 500점

✨ 품질 태그:
   • 태그 없음 []: +15000점 (원본 파일)
   • [!] 검증된 양호: +9000점
   • [f] 수정됨: +7000점
   • [a] 대체: +6000점
   • [T] 번역: +3000점
   • [h] 해킹: +1000점
   • [b] 나쁜 덤프: -5000점
   • [x] 나쁜 체크섬: -10000점

═══════════════════════════════════════════════════════════════════

📋 정리 과정

1️⃣ 스캔
   • 프로그램이 선택된 폴더의 모든 ROM 파일을 분석
   • 게임 이름을 기반으로 중복 식별
   • 각 버전의 우선순위 점수 계산

2️⃣ 분석
   • 게임 이름별로 ROM 그룹화
   • 우선순위 점수별로 버전 정렬
   • 각 게임의 최고 버전 식별

3️⃣ 정리
   • 최고 점수의 버전을 "cleaned" 폴더로 이동
   • 모든 단일 파일(중복 없음)도 "cleaned" 폴더로 이동
   • 열등/중복 버전은 원본 폴더에 남김
   • 상세한 작업 로그 생성

═══════════════════════════════════════════════════════════════════

📁 결과 파일 구조

정리 후:
📂 원본 폴더
   ├── 📄 중복/열등 버전 (여기에 남음)
   └── 📁 cleaned/
       └── 📄 깨끗한 컬렉션: 최고 버전 + 단일 파일 (여기로 이동)

═══════════════════════════════════════════════════════════════════

🎯 사용 팁

💡 첫 사용:
   • 항상 "테스트 모드"를 사용하여 무엇이 일어날지 확인
   • 로그를 확인하여 프로그램의 선택을 검증
   • 최종 정리 전에 컬렉션 백업 생성

🔄 지리적 모드:
   • 이탈리아/유럽 ROM을 선호한다면 "유럽" 선택
   • 주로 미국 컬렉션에는 "미국" 사용
   • 원본 일본 ROM에는 "일본" 사용

📊 로그 해석:
   • ✨ 단일: "cleaned" 폴더로 이동될 고유 파일
   • 📦 최고: "cleaned"로 이동될 중복 그룹의 최고 파일
   • 🔄 남음: 원본 폴더에 남을 중복/열등 파일
   • ⚠️ 멀티디스크: 여러 디스크가 필요한 게임

═══════════════════════════════════════════════════════════════════

⚠️ 중요한 경고

🔐 보안:
   • 프로그램은 파일을 영구적으로 삭제하지 않습니다
   • 깨끗한 컬렉션은 "cleaned" 폴더에 생성됩니다
   • 원본 파일(중복 포함)은 메인 폴더에서 접근 가능하게 남습니다
   • 언제든지 모든 파일을 복구할 수 있습니다

💿 멀티디스크 게임:
   • "파이널 판타지 VII (디스크 1)"과 "파이널 판타지 VII (디스크 2)" 같은 게임
   • 각 디스크는 별도의 게임으로 취급됩니다
   • 필요한 모든 디스크를 유지해야 합니다

🎮 지원되는 형식:
   • .zip, .7z, .rar (압축 아카이브)
   • .iso, .bin, .img (디스크 이미지)
   • .rom, .smd, .smc (직접 ROM)
   • 기타 많은 일반적인 형식
